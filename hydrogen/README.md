# Hydrogen Storefront

Aplicacao headless do projeto usando Shopify Hydrogen.

## Rodar localmente

```bash
npm install
npm run dev
```

## Validar

```bash
npm run lint
npm run typecheck
npm run build
```

## Rotas principais

- `/`
- `/commerce`
- `/about`
- `/dashboard`

## Observacao

Os guias operacionais e a documentacao especifica do storefront estao em `hydrogen/guides/`.
